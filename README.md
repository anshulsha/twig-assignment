# twig assignment

## Getting Started

### Prerequisites

- Node.js (>=12.x)
- npm (>=6.x)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/anshulsha/twig-assignment.git

sh setup.sh

```
2. Now Mongodb and rabbitmq is up and running.
3. Now go to the respective directories and follow their README.md
